# ❤️ Heart Disease Analysis Dashboard

## 📌 Project Overview
This project is a **Heart Disease Analysis Dashboard** built using **Power BI** to analyze healthcare data and identify factors affecting patient survival and heart disease risk.

The dashboard transforms raw patient data into meaningful visual insights that help understand healthcare trends, patient conditions, and survival outcomes.

---

## 🎯 Objectives
- Analyze patient healthcare records
- Identify factors affecting survival rates
- Create interactive visualizations
- Build KPI dashboards for healthcare insights
- Improve analytical and dashboarding skills

---

## 🛠 Tools & Technologies Used
- Power BI
- Excel
- Python
- DAX
- Data Visualization
- Data Cleaning
- Exploratory Data Analysis (EDA)

---

## 📂 Dataset Features
The dataset includes:
- Age
- Diabetes
- High Blood Pressure
- Ejection Fraction
- Serum Sodium
- Platelets
- Creatinine Levels
- Survival/Death Event

Total Records: **299 patients**

---

## 📊 Dashboard Features
- KPI Cards
- Survival vs Death Analysis
- Age Group Analysis
- Disease Distribution Analysis
- Interactive Filters & Slicers
- Medical Indicator Visualization

## 🔍 Key Insights
- Survival percentage is significantly higher than death percentage.
- Certain age groups showed higher survival rates.
- Medical indicators such as serum sodium and ejection fraction may influence survival outcomes.
- Diabetes and high blood pressure patterns were analyzed across different age groups.
